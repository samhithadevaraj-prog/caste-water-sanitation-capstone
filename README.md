# Caste, Water, and Sanitation Inequality in Rural India

This capstone project explores how caste composition may relate to water and sanitation inequalities in rural India. The project uses district-level demographic and infrastructure data to examine patterns of environmental precarity and uneven development.

## Research Focus
- Variation in water and sanitation access across districts
- Relationship between caste composition and infrastructure coverage
- Use of proxy indicators where caste-disaggregated environmental data is unavailable

## Methods
This project begins with data cleaning, merging, and exploratory data analysis in R. Descriptive statistics and visualizations will be used to identify patterns, gaps, and possible inequalities across districts.

## Data
- Census 2011 district-level caste demographics
- Jal Jeevan Mission water access data
- Swachh Bharat sanitation data

## Repository Structure
- `data/raw/` → original files
- `data/processed/` → cleaned datasets
- `scripts/` → R scripts
- `outputs/` → figures and tables
